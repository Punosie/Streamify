# Streamify Dashboard

Streamify Dashboard is a music streaming analytics dashboard that provides insights into user activity, revenue, and content performance for a fictional streaming platform, Streamify. Built using React (Vite) and Chakra UI, the dashboard is fully responsive and visually appealing.

## Installation & Setup

### 1. Clone the repo

```bash
git clone git@github.com:Punosie/Streamify.git
cd Streamify
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start Server

```bash
npm run dev
```
